    [연산자]
    +, -, *, /, **(제곱), %(나머지), ++(증가연산자), --(감소연산자)

    [대입연산자]
    해당변수를 대상으로 연산처리 후 다시 해당 변수에 저장
    =앞에 연산자를 두고 = 뒤에 계산할 값으로 줄여서 표헌
    =양 옆에 변수명이 동일할 때만 적용 가능
    +=, -=, *=, /=, %=

    sum = sum + 변수(1이상의 수)   sum = 변수 + sum

    sum =
    sum + 1

    줄임 sum += 1

    [증감연산자]
    해당변수에 값을 1증가 또는 감소를 시킨다.
    ++, --
    sum = sum + 1(반드시 1을 사용할 때만)
    sum += 1 ===> sum++

    ++sum(선위처리기), sum++(후위처리기), --sum, sum--

    ++sum, sum-- 단독으로 사용할 때는 앞뒤 상관이 없다.

    a=5; b=5;
    tot = ++a + b++; 연산식에 증감연산자가 포함이 되면 ++위치가 중요

    풀이)
    a=a+1   a=6
    tot = a + b   6+5=11
    b=b + 1   b=6

    [비교언산자(관계연산자)]
    두 값을 비교해서 맞으면 true(1), 틀리면 false(0)
    a>b     a가 b보다 크다.(초과)
    a>=b    a가 b보다 크거나 같다.(이상)
    a= 60 && eng >= 60 국어가 60점 이상이고 영어가 60점 이상이면 true
    논리합, ||, a || b : a 와 b 중 하나라도 맞으면 true, 모두 틀리면 false
        중, 또는
        예) kok >= 60 || eng >= 60 국어가 60점이상이거나 영어가 60점 이상이면 true
    부정, ! 현재 결과를 반대로 출력(true -> false, false -> true)

    [문자열 연산]
    문자열 결합(+)
    숫자 + 숫자 = 숫자        5+5=10
    문자열 + 숫자 = 문자열      "국어"+50=국어50
    숫자 + 문자열 = 문자열      50+"점"=50점
    문자열 + 문자열 = 문자열     "안녕"+"하세요"="안녕하세요"

    [조건문(제어문)]
    프로그램의 흐름을 제어하는 명령
    if(조건) { ==>특정 조건에 내용을 추가하고자 할때
    조건을 만족할 때 추가 처리할 부분
    }
    조건처리 또는 만족하지 못했을 때 공통으로 처리

    if(조건) { ==>조건에 따라 처리하는 내용이 다른 경우
    조건을 만족했을 때 처리할 부분
    } else {
    조건을 만족하지 못했을 때 처리할 부분
    }

    } else if(국어>=70점일때) {70점일때, 80점일때 else if 일때는 위에 if문이 없다고 생각
    } else { 80점일때
    }

    swtich문
    두 값이 같은 경우에만 비교(if~else if 비슷한데 관계연산자는 ==만 사용 가능)
    switch(변수명) {
        case 값1 : 값1이 같으면
            break;
        case 값1 : 값2이 같으면
            break;
        default: 만족하는 값이 없을 때 (else)
    }

    a=5;
    switch(a) {
        case 1: a값이 1과 같으면
            console("1입력");
            break;
        case 2: a값이 2와 같으면
            console("2입력");
            break;
        default: a값과 일치하는 내용이 없으면
            console("일치하는 값 없음");
    }
